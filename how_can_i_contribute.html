How can I contribute ?

This project is open-source (Check license section).

Please feel free to email me about bugs, or suggestions on how to improve this project. The code base of this project is relatively small now. Any developer with moderate exposure to the Android SDK and the Java programming language can manipulate the code base with ease. If you woud like to contribute code to this project, then drop me a mail and we can talk.

You could fork this project on GitHub for other mobile platforms. I will be much interested in helping you out for the same.
